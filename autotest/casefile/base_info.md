# deepin-screen-recorder YAML用例生成基础信息文档

## 一、应用基本信息

- **应用名称**: deepin-screen-recorder（截图录屏）
- **包名**: deepin-screen-recorder
- **二进制**: /usr/bin/deepin-screen-recorder
- **Desktop文件**: deepin-screen-recorder.desktop
- **技术栈**: Qt/C++, DTK（Deepin Tool Kit）
- **功能**: 系统截图、录屏、滚动截图、OCR文字识别、AI截图分析、贴图

## 二、启动方式

| 启动方式 | 命令/快捷键 | 说明 |
|---------|-----------|------|
| 截图快捷键 | Ctrl+Alt+A | 进入截图模式 |
| 录屏快捷键 | Ctrl+Alt+R | 进入录屏模式 |
| 窗口截图 | Alt+PrintScreen | 截取当前窗口 |
| 全屏截图 | PrintScreen | 截取全屏 |
| 延时截图 | deepin-screen-recorder -d 3 | 延时3秒截图 |
| 命令行截图 | deepin-screen-recorder -f /path/file.png | 截图保存到指定路径 |
| 贴图 | Alt+P | 贴图模式 |

## 三、应用功能模式

### 3.1 截图模式 (MainWindow::shot)
- 全屏截图、区域选择截图、窗口截图
- 支持标注工具：矩形、椭圆、直线、箭头、画笔、模糊/马赛克、文本
- 支持颜色选择：12种颜色
- 支持撤销操作 (Ctrl+Z)
- 保存选项：PNG/JPG/BMP格式
- 边框效果：阴影、边框、设备边框
- 保存路径：剪切板/桌面/图片/自定义路径

### 3.2 录屏模式 (MainWindow::record)
- 格式：GIF / MP4 / MKV
- 帧率：5/10/20/24/30 fps
- 音频：无/麦克风/系统音频/麦克风+系统音频
- 鼠标选项：无/显示光标/显示点击/光标+点击
- 摄像头：支持开启/关闭
- 显示按键：支持显示/隐藏键盘按键
- 3秒倒计时启动
- 保存路径：视频/桌面

### 3.3 滚动截图模式 (MainWindow::scrollshot)
- 自动滚动/手动滚动
- 支持PNG/JPG/BMP格式
- 滚动过快提示

### 3.4 OCR模式 (MainWindow::ocr)
- 文字识别
- 复制文本/保存为TXT
- 多窗口支持
- 图片预览区域（缩放/拖拽）

### 3.5 贴图模式 (MainWindow::pinscreenshots)
- 截图内容贴图到桌面
- 支持缩放、拖动
- 贴图工具栏（保存/关闭）
- 最多30张贴图
- 系统交互（重启关闭/注销关闭/休眠不关闭）

### 3.6 AI截图功能
- 解释截图、总结截图、翻译文字、问问AI
- AI助手面板交互（上传图片、输入文件）
- 需要uos-ai支持
